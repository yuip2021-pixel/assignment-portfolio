# 과제 포트폴리오 웹사이트 - 디자인 철학

## 선택된 디자인 접근: Modern Minimalism with Playful Accents

### 핵심 원칙
1. **명확한 계층 구조**: 과제들이 동등하게 돋보이면서도 시각적 우선순위가 명확함
2. **공간의 호흡**: 넉넉한 여백으로 각 과제가 독립적인 존재감을 가짐
3. **미묘한 상호작용**: 호버 효과와 부드러운 전환으로 인터랙티브성 표현
4. **조화로운 색상**: 파스텔 톤의 배경과 진한 텍스트의 대비로 가독성 확보

### 색상 철학
- **배경**: 밝고 따뜻한 크림색 (거의 흰색에 가까운 베이지)
- **카드 배경**: 각 과제마다 미묘한 파스텔 톤 (분홍, 하늘색, 노란색, 연두색)
- **텍스트**: 진한 슬레이트 그레이 (높은 명도 대비)
- **강조색**: 부드러운 보라색과 핑크 톤

### 레이아웃 패러다임
- **2x2 그리드**: 정적 작품 (좌상단, 우상단) → 동적 작품 (좌하단, 우하단)
- **카드 기반**: 각 과제를 독립적인 카드로 표현, 그림자로 깊이 표현
- **제목 영역**: 전체 페이지 상단에 포트폴리오 제목과 소개 배치

### 시그니처 요소
1. **부드러운 그림자**: 카드에 미묘한 그림자로 부유감 표현
2. **라운드 코너**: 모든 카드와 요소에 일관된 라운드 처리
3. **호버 애니메이션**: 카드 위에 마우스를 올리면 살짝 올라오는 효과

### 상호작용 철학
- 마우스 호버 시 카드가 부드럽게 상승 (transform: translateY)
- 클릭 시 즉각적인 시각 피드백
- 부드러운 전환 (transition: 300ms ease-out)

### 애니메이션 가이드
- **카드 호버**: 300ms ease-out, scale 1.02 + translateY(-8px)
- **페이드 인**: 초기 로드 시 카드들이 순차적으로 나타남 (stagger 80ms)
- **텍스트 호버**: 색상 변화 200ms ease-out

### 타이포그래피 시스템
- **제목**: 'Poppins' Bold (600-700) - 현대적이고 친근한 느낌
- **부제목**: 'Poppins' Medium (500) - 계층 구조 강화
- **본문**: 'Inter' Regular (400) - 깔끔하고 읽기 쉬운 기본 폰트
- **라인 높이**: 1.6 (충분한 공간감)
